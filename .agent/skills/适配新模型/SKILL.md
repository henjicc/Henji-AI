# 适配新模型

按照制定步骤给痕迹 AI 适配新的模型，首先生成配置文件，然后制定开发计划，接着编写具体代码，最后检查代码是否符合要求。

## 什么时候使用这个技能

- 当用户要求适配新模型的时候

## 如何使用这个技能

当用户要求适配新模型时，按照以下步骤一步步执行
如果用户要求其中的具体某一步骤，直接执行该步骤

1. 生成配置文件
结合当前项目代码以及用户提供的 API 文档，按照 resources/配置模板.yaml 生成新的 yaml 配置文件，参考配置模板使用中文，无需生成注释，但要注意配置模板中的注释说明
在正式生成 yaml 文件前，如果有任何疑问或不确定的事宜，请和用户进行沟通
等用户确认所有信息后，正式生成 yaml 文件，按照模型名称命名，放在 resources/配置文件/ 目录下

2. 制定开发计划
请依据用户提供的 API 文档，自动判断模型供应商，从 resources/适配指南/ 下找到对应的适配指南，如果未找到对应的适配指南，请和用户进行沟通
结合用户提供的 API 文档、上一步生成的 yaml 文件以及对应供应商的模型适配指南，制定一个详细的开发计划提交给用户审查，等待用户确认开发计划，再执行下一步

3. 编写代码
根据开发计划，开始编写代码

4. 检查代码
依据 resources/检查清单.md 逐一检查代码，如果遇到有问题的，请直接修复，直到所有检查项都符合要求
