# Kling-2.6 文生视频

基于 Kling-2.6 先进 AI 模型，通过文本描述生成高质量视频。

## 查询任务状态

提交任务后，可通过统一查询接口查看进度并获取结果。了解如何查询任务状态及获取生成结果，请参阅：[Get Task Details](/cn/market/common/get-task-detail)。

生产环境建议使用 `callBackUrl` 参数接收生成完成的自动通知，而非轮询状态接口。
