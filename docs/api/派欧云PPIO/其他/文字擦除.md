# 文字擦除

文字擦除 API 能将图片中的文字内容擦除干净。

**请求端点**
```
POST https://api.ppinfra.com/v3/remove-text
```

**cURL 请求示例**
```bash
curl --request POST \
  --url https://api.ppinfra.com/v3/remove-text \
  --header 'Authorization: <authorization>' \
  --header 'Content-Type: <content-type>' \
  --data '
{
  "extra": {
    "response_image_type": "<string>"
  },
  "image_file": "<string>"
}
'
```

**响应示例**
```json
{
  "image_file": "<string>",
  "image_type": "<string>"
}
```

## 请求头

**Content-Type** `string` **必需**
*   枚举值：`application/json`

**Authorization** `string` **必需**
*   Bearer 身份验证格式，例如：`Bearer {{API 密钥}}`。

## 请求体

**image_file** `string` **必需**
*   图片二进制的 base64 内容。
*   图片长和宽均不超过 1024 像素。
*   文件最大不超过 30 Mb。

**extra** `object` *额外参数*

*   **response_image_type** `string`
    *   返回的图片格式，默认为 PNG。
    *   枚举值：`png`, `webp`, `jpeg`

## 响应参数

**image_file** `string`
*   图片结果的 Base64 编码内容。

**image_type** `string`
*   图片格式。
*   枚举值：`png`, `webp`, `jpeg`
