# 查找模型

查找模型 API 用于获取可用模型列表。

**请求示例**

```bash
curl --request GET \
  --url https://api.ppinfra.com/v3/model \
  --header 'Authorization: <authorization>' \
  --header 'Content-Type: application/json'
```

**响应示例**

```json
{
  "models": [
    {
      "id": 123,
      "name": "<string>",
      "model_name": "<string>",
      "hash_sha256": "<string>",
      "type": {
        "name": "<string>",
        "display_name": "<string>"
      },
      "categories": {},
      "status": 123,
      "download_url": "<string>",
      "tags": {},
      "cover_url": "<string>",
      "source": "<string>",
      "base_model": "<string>",
      "base_model_type": "<string>",
      "download_url_ttl": 123
    }
  ],
  "pagination": {
    "next_cursor": "<string>"
  }
}
```

## 请求头

| 参数 | 类型 | 是否必需 | 描述 |
| :--- | :--- | :--- | :--- |
| `Content-Type` | string | 是 | 必须为 `application/json`。 |
| `Authorization` | string | 是 | Bearer 身份验证格式，例如：`Bearer {{API 密钥}}`。 |

## 查询参数

| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |
| `filter.visibility` | string | 模型可见性。如果不设置，接口将查询所有类型的模型。枚举值：`public`, `private` |
| `filter.types` | string | 模型类型。枚举值：`checkpoint`, `lora`, `vae`, `controlnet`, `upscaler`, `textualinversion` |
| `filter.is_sdxl` | boolean | 是否返回 SDXL 模型。 |
| `filter.is_inpainting` | boolean | 是否返回 Inpainting 模型。 |
| `pagination.limit` | string | 每页返回的模型数，范围 (0, 100]。 |
| `pagination.cursor` | string | 指定从第几个模型开始返回。如果为空，则表示从头开始获取。通常赋值为响应包中返回的 `next_cursor` 字段值来获取下一页的内容。 |

## 响应参数

### `models` (object[])

模型列表。

| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |
| `id` | integer | 模型的 ID。 |
| `name` | string | 模型名称。 |
| `model_name` | string | API 中使用的模型名称。 |
| `hash_sha256` | string | 模型文件的哈希值。 |
| `type` | object | 模型类型。 |
| `type.name` | string | 模型类型名称。 |
| `type.display_name` | string | 模型类型显示名称。 |
| `categories` | [string] | 模型类别。 |
| `status` | integer | 模型状态：0 表示不可用，1 表示可用。 |
| `download_url` | string | 模型下载 URL。 |
| `tags` | [string] | 模型标签。 |
| `cover_url` | string | 模型封面图像 URL。 |
| `source` | string | 模型的来源。 |
| `base_model` | string | 模型的基础模型。 |
| `base_model_type` | string | 模型的基础模型类型。 |
| `download_url_ttl` | integer | 下载 URL 的过期时间（以秒为单位），默认为 1 天。 |

### `pagination` (object)

分页信息。

| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |
| `next_cursor` | string | 下一页请求的起始位置。 |
