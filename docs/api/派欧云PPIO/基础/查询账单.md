# 查询账单

## 接口说明
查询按量计费账单信息。

**请求方法:** GET

**请求地址:**
```
https://api.ppinfra.com/openapi/v1/billing/bill/list
```

## 请求示例

```bash
curl --request GET \
  --url https://api.ppinfra.com/openapi/v1/billing/bill/list \
  --header 'Authorization: <authorization>'
```

## 请求头

| 参数 | 类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| Authorization | string | 是 | Bearer 身份验证格式，例如：`Bearer {{API 密钥}}`。 |

## 查询参数

| 参数 | 类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| cycleType | string | 是 | 账单的统计粒度。取值：`Hour`（按小时）、`Day`（按天）、`Week`（按周）、`Month`（按月）。 |
| productCategory | string | 是 | 产品类型。取值：`llm`（大语言模型）、`gen_api`（图片/视频模型）。 |
| startTime | string | 否 | 要查询的账单的开始时间。 |
| endTime | string | 否 | 要查询的账单的结束时间。 |

## 响应参数

**响应体结构:**
```json
{
  "bills": [
    {
      "userId": "<string>",
      "startTime": "<string>",
      "endTime": "<string>",
      "billingMethod": "<string>",
      "productName": "<string>",
      "category": "<string>",
      "ownerID": "<string>",
      "billNum0": "<string>",
      "billNum1": "<string>",
      "basePrice0": "<string>",
      "basePrice1": "<string>",
      "discountPrice0": "<string>",
      "discountPrice1": "<string>",
      "amount": "<string>",
      "voucherAmount": "<string>",
      "payAmount": "<string>",
      "pricePrecision": "<string>"
    }
  ]
}
```

**bills 对象字段说明:**

| 字段 | 类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| userId | string | 是 | 用户账号 ID。 |
| startTime | string | 是 | 账单的开始时间。格式为 Unix 时间戳。 |
| endTime | string | 是 | 账单的结束时间。格式为 Unix 时间戳。 |
| billingMethod | string | 是 | 计费方式。取值为 `1` 表示按量计费。 |
| productName | string | 是 | 产品名称。 |
| category | string | 是 | 产品类型。 |
| ownerID | string | 是 | 实例 ID。 |
| billNum0 | string | 是 | 产品类型为 `llm` 时，表示输入 Token 数量。 |
| billNum1 | string | 是 | 产品类型为 `llm` 时，表示输出 Token 数量。 |
| basePrice0 | string | 是 | 原价。 |
| basePrice1 | string | 是 | 该参数无意义。 |
| discountPrice0 | string | 是 | 单价。产品类型为 `llm` 时，表示输入 Token 单价。 |
| discountPrice1 | string | 是 | 产品类型为 `llm` 时，表示输出 Token 单价。 |
| amount | string | 是 | 总价。 |
| voucherAmount | string | 是 | 代金劵抵扣费用。 |
| payAmount | string | 是 | 现金支付费用。 |
| pricePrecision | string | 是 | 价格精度。单价（元）= 单价 / 10000 / pricePrecision。 |
