# 获取账户信息

> 来源: https://ppio.com/docs/models/reference-get-user-info

「获取账户信息 API」主要用于查询账户余额等信息。

**请求方法:** GET

**请求地址:** `https://api.ppinfra.com/v3/user`

## 请求示例

```bash
curl --request GET \
  --url https://api.ppinfra.com/v3/user \
  --header 'Authorization: <authorization>' \
  --header 'Content-Type: <content-type>'
```

## 响应示例

```json
{
  "credit_balance": 123
}
```

## 请求头

| 参数名 | 类型 | 是否必需 | 说明 |
| :--- | :--- | :--- | :--- |
| `Content-Type` | string | 是 | 固定为 `application/json`。 |
| `Authorization` | string | 是 | Bearer 身份验证格式，例如：`Bearer {{API 密钥}}`。 |

## 响应参数

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `credit_balance` | number | 账户余额信息。 |
