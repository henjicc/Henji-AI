# 对象擦除

对象擦除 API 能将图片中的指定区域擦除干净。

**请求示例**

```bash
curl --request POST \
  --url https://api.ppinfra.com/v3/cleanup \
  --header 'Authorization: <authorization>' \
  --header 'Content-Type: <content-type>' \
  --data '
{
  "extra": {
    "response_image_type": "<string>"
  },
  "image_file": "<string>",
  "mask_file": "<string>"
}
'
```

**响应示例**

```json
{
  "image_file": "<string>",
  "image_type": "<string>"
}
```

## 请求头

### Content-Type
*   **类型**: `string`
*   **必需**: 是
*   **枚举值**: `application/json`

### Authorization
*   **类型**: `string`
*   **必需**: 是
*   **描述**: Bearer 身份验证格式，例如：`Bearer {{API 密钥}}`。

## 请求体

### image_file
*   **类型**: `string`
*   **必需**: 是
*   **描述**: 原始图片的 base64 编码内容，支持的最大分辨率小于 16 megapixels，文件最大不超过 30 Mb。

### mask_file
*   **类型**: `string`
*   **必需**: 是
*   **描述**: 遮罩图片的 base64 编码内容，支持的最大分辨率小于 16 megapixels，文件最大不超过 30 Mb。

### extra
*   **类型**: `object`
*   **描述**: 额外参数。

#### extra.response_image_type
*   **类型**: `string`
*   **描述**: 返回的图片格式，默认为 PNG。
*   **枚举值**: `png`, `webp`, `jpeg`

## 响应参数

### image_file
*   **类型**: `string`
*   **描述**: 图片结果的 Base64 编码内容。

### image_type
*   **类型**: `string`
*   **描述**: 图片格式。
*   **枚举值**: `png`, `webp`, `jpeg`
