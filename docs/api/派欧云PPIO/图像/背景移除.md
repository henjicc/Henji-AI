# 背景移除

背景移除 API 能移除图片中的背景内容。

## 请求

**端点**
```
POST https://api.ppinfra.com/v3/remove-background
```

**cURL 示例**
```bash
curl --request POST \
  --url https://api.ppinfra.com/v3/remove-background \
  --header 'Authorization: <authorization>' \
  --header 'Content-Type: <content-type>' \
  --data '
{
  "extra": {
    "response_image_type": "<string>"
  },
  "image_file": "<string>"
}
'
```

## 请求头

| 参数 | 类型 | 是否必需 | 描述 |
| :--- | :--- | :--- | :--- |
| `Content-Type` | string | 是 | 枚举值: `application/json` |
| `Authorization` | string | 是 | Bearer 身份验证格式，例如：`Bearer {{API 密钥}}`。 |

## 请求体

| 参数 | 类型 | 是否必需 | 描述 |
| :--- | :--- | :--- | :--- |
| `image_file` | string | 是 | 图片二进制的 base64 内容，图片长和宽均不超过 1024 像素，文件最大不超过 30 Mb。 |
| `extra` | object | 否 | 额外参数。 |
| `extra.response_image_type` | string | 否 | 返回的图片格式，默认为 PNG。枚举值: `png`, `webp`, `jpeg` |

## 响应参数

**响应示例**
```json
{
  "image_file": "<string>",
  "image_type": "<string>"
}
```

| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |
| `image_file` | string | 图片结果的 Base64 编码内容。 |
| `image_type` | string | 图片格式。枚举值: `png`, `webp`, `jpeg` |
